## 프로젝트 소개

COVID-19로 위축되었던 스포츠 활동이 다시 활성화 되며, 많은 체육관들이 다시 운영되고 있습니다.

이러한 요즘, 스포츠 활동을 새롭게 시작하기에 앞서, 정보가 부족하여 불편하진 않으셨나요?

위치기반 체육관 검색부터 예약, 대여용품 선택 그리고 선호하는 체육관 찜하기까지 한 번에 이용해 보세요!
